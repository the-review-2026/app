# The Review (Quest Mode)

カード方式ではなく、クエストとストーリーで進める復習アプリです。  
`index.html` をブラウザで開くだけで使えます。

## 画面構成

- `クエスト`: ユニットごとの問題をクリアしてXP獲得
- `ストーリー`: クエスト達成で解放されるシナリオ
- `マイページ`: 正答率、読了数、連続日数
- `設定`: テーマカラー、通知設定、進捗リセット

## 使い方

1. `クエスト` で解放済みノードを選択
2. 問題に回答し、ユニットを順番にクリア
3. `ストーリー` で解放されたシーンを進行
4. `マイページ` で学習状況を確認

## 保存データ

- `localStorage` に進捗を保存します
- 保存キー: `the-review-quest-v1`

## ファイル構成

- `index.html`: 画面構造
- `styles.css`: クエストUIと全体デザイン
- `app.js`: クエスト進行、ストーリー解放、進捗保存
- `assets/the-review-logo.png`: ヘッダーロゴ
